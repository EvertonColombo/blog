# Everton Eduardo Colombo - Blog

Personal blog built with Quarto.

## Local Development

1. Install Quarto.
2. Clone the repository.
3. Run preview:
   ```bash
   quarto preview
   ```

## Deployment

This repository uses **GitHub Actions** to automatically publish changes.
Just push to the `main` branch, and the workflow will build and deploy the site to the `gh-pages` branch.
